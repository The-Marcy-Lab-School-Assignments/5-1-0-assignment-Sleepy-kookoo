# Question
In a code snippet, demonstrate a `User` class with at least one:
- class property
- class method
- instance property
- instance method

# Answer
```js
class User {
  constructor (name, age) {
    // Instance properties
    this.name = name;
    this.age = age;
  }

  // Instance method
  greet() {
    return `Hello, my name is ${this.name} and I am ${this.age} years old.`;
  }
}

let user1 = new User("America", 20);
console.log(user1.greet());

```
